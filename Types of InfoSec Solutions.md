| Система                                                    | Описание                                                                                                                                                      | Примеры                                                                |
| ---------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| **IRP** (Incident Response Platform)                       | Платформа для автоматизации процессов реагирования на инциденты ИБ и организации совместной работы сотрудников (включая оркестрацию и функционал SOAR).       | • **TheHive** (с интеграцией Cortex)  <br>• **IBM Resilient**          |
| **EDR** (Endpoint Detection and Response)                  | Средство для выявления и анализа вредоносной активности на конечных устройствах (ПК, серверы), обеспечивающее автоматическую изоляцию и сбор цифровых следов. | • **CrowdStrike Falcon**  <br>• **Kaspersky EDR**                      |
| **SIEM** (Security Information and Event Management)       | Решение для сбора, корреляции и анализа событий ИБ в реальном времени, позволяющее выявлять аномалии и формировать оповещения.                                | • **Splunk Enterprise Security**  <br>• **QRadar (IBM)**               |
| **TI/TH-платформа** (Threat Intelligence / Threat Hunting) | Система для проактивного поиска признаков компрометации в инфраструктуре с применением актуальных разведывательных данных об угрозах и гипотез атак.          | • **MISP** (Open Source TI)  <br>• **Recorded Future**                 |
| **DLP** (Data Loss Prevention)                             | Комплекс защиты от утечек конфиденциальных данных за пределы контролируемого периметра (мониторинг каналов: email, USB, облачные сервисы, печать).            | • **InfoWatch Traffic Monitor**  <br>• **Symantec DLP**                |
| **WAF** (Web Application Firewall)                         | Межсетевой экран уровня веб-приложений, который анализирует и фильтрует входящий трафик, блокируя SQL-инъекции, межсайтовый скриптинг (XSS) и другие атаки.   | • **AWS WAF**  <br>• **ModSecurity** (с правилами OWASP Core Rule Set) |
