# Knowledge-Base

Данный репозиторий - это мои личные учебные заметки. Информация собрана из открытых источников и предназначена только для самообучения и легального тестирования своей инфраструктуры.
Несанкционированное применение описанных методов к чужим ресурсам преследуется по закону. Вся ответственность за использование информации лежит на злоумышленнике, а не на авторе заметок.
