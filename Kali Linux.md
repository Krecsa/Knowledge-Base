# Шпаргалка по каталогам Linux

| **Описание назначения каталога**                             | **Где лежит** | **Примеры**                              |
| ------------------------------------------------------------ | ------------- | ---------------------------------------- |
| Базовые исполняемые файлы (команды) для всех пользователей.  | **`/bin`**    | `ls`, `cp`, `bash`                       |
| Системные утилиты для администрирования (требуют `sudo`).    | **`/sbin`**   | `fdisk`, `iptables`, `init`              |
| Библиотеки для программ из `/bin` и `/sbin`.                 | **`/lib`**    | `libc.so` (библиотека языка C)           |
| Аналог `Program Files`).                                     | **`/usr`**    | `/usr/bin`, `/usr/lib`, `/usr/share`     |
| Конфигурационные файлы системы и программ.                   | **`/etc`**    | `/etc/passwd`, `/etc/network/interfaces` |
| Переменные данные (логи, кэш, почта).                        | **`/var`**    | `/var/log`, `/var/www` (веб-серверы)     |
| Временные файлы (очищаются после перезагрузки).              | **`/tmp`**    |                                          |
| Файлы загрузчика (ядро, initramfs, GRUB).                    | **`/boot`**   | `vmlinuz-5.15.0-76-generic`              |
| Файлы устройств (жёсткие диски, USB, виртуальные терминалы). | **`/dev`**    | `/dev/sda`, `/dev/tty1`                  |
| Виртуальная ФС с информацией о процессах и системе.          | **`/proc`**   | `/proc/cpuinfo`, `/proc/meminfo`         |
| Информация о ядре и устройствах.                             | **`/sys`**    |                                          |
| Домашние каталоги пользователей.                             | **`/home`**   | `/home/alice`, `/home/bob`               |
| Домашний каталог суперпользователя (не путать с `/`!).       | **`/root`**   |                                          |
| Временные точки монтирования (например флешек).              | **`/mnt`**    | **`/mnt/usb`**                           |
| Автомонтирование устройств (CD/DVD, USB).                    | **`/media`**  | `/media/usb-drive`                       |
| Дополнительное ПО (часто проприетарное)                      | **`/opt`**    | `/opt/google/chrome`                     |
## **Сравнение с Windows и macOS**

| Назначение                  | Linux              | Windows                           | macOS                               |
| --------------------------- | ------------------ | --------------------------------- | ----------------------------------- |
| Домашняя папка пользователя | `/home/user`       | `C:\Users\user`                   | `/Users/user`                       |
| Глобальные конфиги          | `/etc`             | `C:\ProgramData`, реестр          | `/etc` → `/private/etc`             |
| Системные команды           | `/bin`, `/usr/bin` | `C:\Windows\System32`             | `/bin`, `/usr/bin` (симлинки)       |
| Блочные устройства          | `/dev/sda1`        | `\\.\PhysicalDrive0`              | `/dev/disk0s1`                      |
| Съёмные носители            | `/mnt`, `/media`   | `D:\`, `E:\`                      | `/Volumes`                          |
| Временные файлы             | `/tmp`             | `C:\Windows\Temp`                 | `/tmp` → `/private/tmp`             |
| Логи                        | `/var/log`         | `C:\Windows\Logs`, `Event Viewer` | `/var/log` → `/private/var/log`     |
| Программное обеспечение     | `/opt`             | `C:\Program Files`                | `/Applications` (GUI), `/opt` (CLI) |
| Пользовательские настройки  | `~/.config`        | `%APPDATA%`                       | `~/Library/Preferences`             |
| Кэш пользователя            | `~/.cache`         | `%LOCALAPPDATA%\Temp`             | `~/Library/Caches`                  |

## **Следует запомнить**

`/var/log` - Здесь можно найти логи
**`/etc/passwd`** - Здесь хранится основная информация о пользователях и логины
**sudo** - Временно дает права root, записывает все действия в `/var/log/auth.log`
